Using the Finances API 

[Retrieving payout information](payout-info.html)

[Retrieving other monetary transactional data](transaction-info.html)

[Setting up split payouts for Mainland China sellers](split-payouts.html)

The [Finances API](/api-docs/sell/finances/overview.html) allows sellers to review and track financial information about their eBay account.

Sellers can view the status of one or more payouts, determine how many transactions have been made within a specific time, determine the amount of fees and charges that have been paid to eBay, and view the total amount of funds available for a payout that have not been processed yet.

See the following topics for more information on using the Finances API :

* [Retrieving Payout Information](payout-info.html) - The Finances API allows sellers to review and track information relating to seller payouts. They can also view pending funds that have not been distributed through a seller payout.
* [Retrieving Other Monetary Transactional Data](transaction-info.html) - The Finances API allows sellers to review and track information relating to monetary transactions, as well as receive the details and charges associated with a Transfer transaction.
* [Setting up Split Payouts for Mainland China Sellers](split-payouts.html) - When used concurrently with the [Account v2 API](/api-docs/sell/account/v2/static/overview.html), sellers in Mainland China can use the Finances API to retrieve split payout information.
