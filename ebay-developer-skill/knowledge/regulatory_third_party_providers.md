# Regulated Third Party Providers (TPP) Guide

Source: https://developer.ebay.com/develop/guides/regulated-third-party-providers-tpp

This guide covers requirements and processes for Regulated Third Party Providers (TPPs) integrating with eBay APIs under EU/UK financial services regulations.

---

## Overview

Regulated TPPs can register with eBay and obtain client details and tokens required to make API calls under financial services regulatory frameworks.

---

## Register with eBay

TPPs must complete the registration process to obtain:
- Client credentials
- Access tokens
- Required API permissions

[Registration Guide](https://developer.ebay.com/develop/guides/regulated-third-party-providers-tpp/register-with-ebay)
