Expo is a framework that makes developing Android and iOS apps easier. Our framework provides file-based routing, a standard library of native modules, and much more. Expo is open source with an active community onGitHubandDiscord.

We also makeExpo Application Services (EAS), a set of services that complement the Expo framework in each step of the development process.
